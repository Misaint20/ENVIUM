# Security Policy

## Supported Versions

Currently, only the latest major version of **Envium** receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this zero-dependency library very seriously. 

If you discover any security-related issues, please **DO NOT** create a public GitHub Issue.
Instead, responsibly disclose the vulnerability by sending an email to the repository owner or using the GitHub Security Advisories feature (if enabled).

### What to include in your report
- A description of the vulnerability and its impact.
- Steps to reproduce the issue.
- (Optional) A Proof of Concept (PoC) script or project.

You will receive a response within 48 hours. If the vulnerability is accepted, we will coordinate a fix and an advisory before it goes public.
